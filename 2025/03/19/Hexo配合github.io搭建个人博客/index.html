Hexo配合github.io搭建个人博客

2025-03-19

1.创建github仓库

在创建github仓库的时候，有一些讲究；仓库的名称必须是<username> + github.com组合
例如我的就是： Junglezt.github.com

2.在本地搭建hexo

hexo是一个轻量级，专门为个人建立网站的博客框架

2.1 安装 hexo

hexo依赖node环境，首先我们需要安装node
参考node官网: https://nodejs.org/zh-cn/download

下载后使用npm安装hexo网站
参考hexo官网: https://hexo.io/zh-cn/

hexo有一些常用的命令

2.2 构建并部署 hexo

访问即可看到默认的hexo网站: http://localhost:4000/

3. 配置hexo本地更新github.io托管

参考官网: https://hexo.io/docs/github-pages

安装hexo-deployer-git插件

修改_config.yml

官网默认的配置是使用token的方式验证，我没有认证成功，上述可以看出我使用的是ssh连接，步骤如下

执行后一直回车，默认会在用户的家目录中创建一个.ssh的目录，其中有id_rsa、id_rsa.pub文件
Windows: c:\users\<username>\.ssh\
Linux: /home/<username>/.ssh

接着登录到github后台，头像 -> Setting -> SSH and GPG keys -> New SSH key
title起一个容易记忆名字: Junglezt.github.io
复制id_rsa.pub文件中的所有内容到Key，点击ADD SSH Key

如果添加成功，可以使用ssh命令测试是否添加成功

如果上述配置成功，执行hexo d即可同步到github.io个人网站
